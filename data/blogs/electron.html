我是将electron-quick-start clone 下来的基础上开发的看到目录里面有三个主要的js文件main.js/renderer.js/preload.js

main.js

Modules to control application life and create native browser window

node本地客户端的所有操作

renderer.js

This file is required by the index.html file and will be executed in the renderer process for that window. No Node.js APIs are available in this process because `nodeIntegration` is turned off. Use `preload.js` to selectively enable features needed in the rendering process.

index.html文件中执行的脚本,可以自由的执行Dom操作

preload.js

All of the Node.js APIs are available in the preload process. It has the same sandbox as a Chrome extension.

最快速预加载脚本, 最预先加载js文档.
